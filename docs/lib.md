# Frontend Libraries

## State Management (Pinia)

Quản lý trạng thái toàn cục như danh sách phòng, thông tin người dùng đăng nhập. Dễ học, tích hợp tốt với Nuxt.

## Data Fetching (TanStack Query / Vue Query)

Quản lý gọi API, caching, loading state và tự động refetch dữ liệu như danh sách phòng, hóa đơn.

## UI Components (shadcn-vue)

Cung cấp các component UI hiện đại như Table, Form, Dialog, phù hợp cho trang quản trị.

## Form Handling (VeeValidate + Zod)

Xây dựng và validate form nhập liệu một cách an toàn, hỗ trợ TypeScript và tái sử dụng schema.
